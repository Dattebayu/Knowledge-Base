## Манипулирование с базой данных

- Создание базы данных:

```SQL

CREATE DATABASE shop;

```

`CREATE DATABASE` - команда для создания.

`shop` - название базы данных.

- Удаление базы данных:

```SQL

DROP DATABASE shop;

```

`DROP DATABASE` - команда для удаления.

`shop` - название базы данных.
