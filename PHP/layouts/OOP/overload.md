## Перегрузка

Специальное свойство классов, когда мы в дочернем классе записываем метод, с таким же именем и он перезаписывает метод главного класса:

Если нужно сделать отработку именно родительского метода используется следующая запись:

```php

class Second extends First {
  public function start() {
    parent::stars();
  }
}

```
