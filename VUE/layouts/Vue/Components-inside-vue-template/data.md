### data

Специальный объект, в который мы возвращаем переменные, которые будет использовать в шаблонах.

---

Пример:

В данном примере мы создаем переменную и потом выводим ее на экран, через шаблон:

```javascript

<template>
  <div>
    {{ counter }}
  </div>
</template>

<script>
  export default {
    data() {
      return {
        counter: 1
      }
    }
  }
</script>

```
